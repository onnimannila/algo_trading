# Alpaca Trading Algorithm

Repo for testing Python codes and connecting to Alpaca Trading API. No real money is used. Repo for testing Python codes and connecting to Alpaca Trading API. No real money is used. 

The idea is to make a simple trading algorithm that is based on trend-following strategy. 
This is chosen for simplicity. In order to get a deeper dive to the markets one would either have to have more information on markets or larger calculation capacity.

## Steps for the algorithm:
1. connect to Alpaca trading API (paper trading)
2. fetch market data for traded instruments
3. execute rule-based calculations
4. execute trades based on trading signals
5. implement risk management calculations for the portfolio (VaR, Sharpe etc.)
6. adjust strategy based on portfolio composition
7. create a report of the algo performance
